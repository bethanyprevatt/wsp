Project Analysis

Designer: Bethany Prevatt
Client: Tiffany & Co.
Website: www.tiffany.com

Site Intent

The intent of the project design is a redesign of Tiffany and Co.'s website. The redesign is intended to create a responsive design to allow users to view their website on any size device.

Competitive Analysis

Zales

URL: www.zales.com
Summary: Zales is a common jewelry store design, however it is not responsive.

Jared

URL: www.jared.com
Summary: Jared's website is a little different in design than the normal jewelry website. This website is not responsive in design.

ICE

URL: www.ice.com
Summary: Ice's website is nicely laid out and easy to navigate, but it is not responsive.
